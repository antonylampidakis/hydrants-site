Η HydrantsNearby είναι μια mobile εφαρμογή που σας επιτρέπει να εντοπίζετε πυροσβεστικούς κρουνούς σε διαδραστικό χάρτη και να σας καθοδηγεί σε αυτούς. Προσφέρει γρήγορη αναζήτηση, εύκολη πλοήγηση,
φίλτρα και χρήσιμες πληροφορίες για τον κάθε κρουνό.
